# 🧮 Basic Calculator

A simple web calculator using **HTML**, **CSS**, and **JavaScript** for basic arithmetic operations.

## 🚀 Features
- Add, Subtract, Multiply, Divide
- Clear and Backspace
- Responsive design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📦 How to Use
1. Clone/download the repo  
2. Open `index.html` in your browser
